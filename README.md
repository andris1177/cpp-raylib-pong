# cpp-raylib-pong

## install cmake

### windows
Tölsd le és telepísd
https://cmake.org/download/

### linux

ubuntu/debian
``` bash
sudo at-get install cmake
```
arch
``` bash
sudo pacmen -S cmake
```

## linux/windows

clone
``` bash
git clone https://github.com/andris1177/cpp-raylib-pong.git
```

cmake
``` bash
cd cpp-raylib-pong
cmake .
```

make
``` bash
make
```

futtatás
``` bash
./main
```
